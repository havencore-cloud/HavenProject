# Haven Bot Project

## Overview
Haven Bot is designed to empower wealth redistribution by ethically trading cryptocurrency.

## Structure
- `Docs/` - Organizational documents
- `Bot/` - Core bot code
- `Utilities/` - Setup and requirements
- `Tests/` - Test files

## Quick Start
1. Clone repo
2. Install packages
3. Configure `config.py`
4. Run `main.py`
